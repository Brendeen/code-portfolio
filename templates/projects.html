Viper's Venom Tavern

Viper's Venom Tavern, the number one place to buy, sell and trade cards! VVT for short is a E-commerce website made card players looking to expand or sell their collection. User's create an account with a username, email and password. From there you can browse the store, put up your own cards for sale, and view the about page. VVT is still a work in progress, with new features being implemented and reworked.

Full stack web application made with Next.Js and Django Rest framework, styled using Tailwind css, CRUD database using Postgres, and deployed with Netlify.

Text Adventure

Text Adventure is a throw back to old school text games from the 70's. Take on the role of an adventurer searching for the mystical Idol of Power! In this game, user's must manage their health and stamina while traversing through the many environments to find the Idol. Solve puzzles, fight creatures, and make it out alive! Each play through will be unique, as each environment has different scenarios that could play out. Do you have what it takes to survive the dangerous jungle of Anradesha?

Command line text game programmed using Python and styled using rich.
